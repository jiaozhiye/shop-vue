## API

### Anchor

| 参数   | 说明                       | 类型   | 默认值 |
| ------ | -------------------------- | ------ | ------ |
| endVal | 计数结束时的数值，必要参数 | number | -      |
| delay  | 动画延迟时间，单位 ms      | number | 0      |

### 示例

```bash
# template
<template>
  <count-up :endVal="2020" />
</template>
```
